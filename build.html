Obstacle Avoidance Robot

Components: Arduino Uno, HC-SR04 Ultrasonic Sensor, L298N Motor Driver, DC Motors

This robot detects obstacles and automatically changes direction.

  • Ultrasonic VCC → 5V, GND → GND
  • TRIG → Pin 9, ECHO → Pin 10
  • Motor Driver IN1–IN4 → Pins 4,5,6,7
  • Connect motors to driver output
  • Upload obstacle avoidance code
