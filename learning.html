Arduino UNO

Arduino Uno is a beginner-friendly microcontroller board based on the ATmega328P. It is widely used for building electronics and IoT projects because it is simple, affordable, and easy to program.

Pins: 5V, GND, A0-A5, D0-D13

ESP8266 (NodeMCU)

ESP8266 is a low-cost Wi-Fi microcontroller used in IoT projects. It has a built-in TCP/IP stack, allowing devices to connect directly to the internet. It operates at 3.3V and can be programmed using Arduino IDE, MicroPython, or Lua. It supports GPIO pins, PWM, I2C, SPI, and serial communication. It is widely used in smart home systems, IoT automation, and remote monitoring applications.

Pins: D0-D8, 3V3, GND, VIN

Proximity Sensor

A proximity sensor detects nearby objects without physical contact. Most modules use infrared (IR) light to detect reflected signals from objects. It outputs a digital signal when an object is within a certain range. Commonly used in obstacle detection robots, automatic doors, and object counting systems.

Pins: VCC, GND, OUT

Ultrasonic Sensor

An ultrasonic sensor measures distance using high-frequency sound waves. It sends an ultrasonic pulse and calculates the time taken for the echo to return. Distance is calculated using time and speed of sound. Widely used in obstacle avoidance robots, parking sensors, and distance measurement systems.

Pins: VCC, Trig, Echo, GND

PIR Sensor

A PIR (Passive Infrared) sensor detects motion by sensing changes in infrared radiation emitted by warm objects like humans or animals. It does not emit radiation; it only detects it. Commonly used in motion alarms, automatic lights, and security systems.

Pins: VCC, OUT, GND

Gas Sensor (MQ-2)

A gas sensor detects the presence of gases such as LPG, methane, carbon monoxide, and smoke. It works by measuring changes in resistance when exposed to gas. Most MQ-series sensors provide analog output. Used in gas leakage detection and air quality monitoring systems.

Pins: VCC, GND, A0, D0

LDR Sensor

An LDR (Light Dependent Resistor) is a light-sensitive resistor whose resistance decreases when light intensity increases. It is an analog sensor. Commonly used in automatic street lights, light intensity monitoring, and brightness control systems.

Output: Analog

Soil Moisture Sensor

A soil moisture sensor measures the water content in soil. It works by measuring electrical resistance or capacitance between two probes inserted into the soil. It helps determine whether soil is dry or wet. Commonly used in automatic irrigation and smart agriculture systems.

Pins: VCC, GND, A0, D0

Rain Drop Sensor

A rain drop sensor detects water droplets or rainfall using a conductive plate. When water falls on the plate, conductivity increases and the sensor gives a signal. It is used in rain detection systems, automatic wipers, and weather monitoring systems.

Pins: VCC, GND, A0, D0

RFID Sensor

An RFID sensor (Radio Frequency Identification) uses radio waves to read data stored in RFID tags or cards. It consists of a reader and a tag. When the tag comes near the reader, it transmits its unique ID. Commonly used in access control systems, attendance systems, and contactless payment systems.

Protocol: SPI

Fire Sensor

A fire sensor detects flame or fire using infrared light detection. It senses specific wavelengths emitted by flames. It provides a digital output when flame is detected. Used in fire alarm systems and safety monitoring systems.

Pins: VCC, GND, OUT

Touch Sensor

A touch sensor detects human touch using capacitive sensing. When a finger touches the sensor surface, it changes capacitance and triggers an output signal. It works like a digital button but without mechanical parts. Used in touch lamps, control panels, and smart switches.

Pins: VCC, GND, OUT

DHT11 Sensor

A humidity sensor measures the amount of moisture present in the air. Many modules (like DHT series) also measure temperature. It provides either digital or analog output. Used in weather stations, smart farming, air quality monitoring, and climate control systems.

Pins: VCC, Data, GND

Microwave Sensor

A microwave motion sensor detects movement using microwave radar signals. It sends out microwave pulses and measures reflected signals to detect motion. It can detect movement through thin walls and glass. Used in security systems and motion detection lighting.

Pins: VCC, OUT, GND

Time of Flight Sensor

A Time of Flight sensor measures distance by calculating the time taken by a light pulse (usually infrared laser) to travel to an object and return. It provides highly accurate distance measurement, especially at short ranges. Used in robotics, gesture recognition, and 3D mapping systems.

Protocol: I2C

Breadboard

Used for prototyping circuits without soldering.
